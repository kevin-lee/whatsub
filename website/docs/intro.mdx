Whatsub is an open-source easy to use CLI tool for subtitles.

![Whatsub Logo](/img/poster.png)

It can
* convert text files charset.
* convert a `SAMI` (`SMI`) subtitle file to an `SRT` file as well as converting `SRT` to `SMI`.
* re-sync the subtitles which are out of sync.
